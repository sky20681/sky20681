Markdown简介

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。
Markdown 语言在 2004 由约翰·格鲁伯（英语：John Gruber）创建。
Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。
Markdown 编写的文档后缀为 .md, .markdown。

Markdown语法

1.如何换行
方法一：
末尾添加两个空格
示例：
这段换末尾有两个空格
这段话末尾没有两个空格
这段话是参照物
可以看到，方法一没用
方法二：
在段尾使用一个空行

参考资料：菜鸟教程：地址：https://www.runoob.com/markdown/md-paragraph.html
示例：
这是一句话，它下面有一行空行

这是一句话，它上面有一行空行
2.如何在每句话前加个小点
方法：使用※号，+号或者-号在加个空格后面跟句子
示例：

  • 这句话前面是个*号加一个空格
  • 这句话前面是个+号加一个空格
  • 这句话前面是个-号加一个空格

3.如何添加超链接，链接是文字但她是个链接
示例：这是个文字超链接
即把插入图片的感叹号给去掉
有感叹号是个图片地址
没有感叹号是个链接
4.指定图片的高和宽
Markdown 还没有办法指定图片的高度与宽度，如果你需要的话，你可以使用普通的 img标签
示例图片大小：

下面是20%，40%，80%，100%,200%,500%示例






可以看到最大不能超过这个主题规定的文章的宽度
5.如何加粗文本
方法：在文本前后加两个*号
示例：
这句话前后有两个星号
这句话前后有三个
三个星号是加斜加粗
