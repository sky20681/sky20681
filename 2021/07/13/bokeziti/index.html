MAtery主题如何更改字体

麻溜的开始吧

默认的就好看，没事别瞎折腾

准备工作

---

挑选一款心仪的字体
这里我挑了几款好看的

然后拿告白体做演示把🐶

准备工作

接着在hexo根目录的source中新建font文件夹，把下载好的ttf格式的字体放进去，以防万一，在根目录的网站_config.yml配置文件中，跳过这整个文件夹渲染，如下：

新建CSS文件

文字的格式需要CSS来调整，在matery主题/source/css文件夹下新建一个diy.css文件

放入如下代码：

/* Here is your diy css styles. */
/* 自定义css存放处*/
@font-face {    
    font-family: 'myfont';    
    src: url('https://cdn.jsdelivr.net/gh/sky20681/sky20681/font/Aa望江南.ttf'),
    url('https://cdn.jsdelivr.net/gh/sky20681/sky20681/font/告白体.ttf'),
    url('https://cdn.jsdelivr.net/gh/sky20681/sky20681/font/Aa初吻.ttf'),
    url('https://cdn.jsdelivr.net/gh/sky20681/sky20681/font/Aa奶糖初恋体.ttf'),
    url('https://cdn.jsdelivr.net/gh/sky20681/sky20681/font/Aa甜茶.ttf');
 }
 
 body{
     font-family: 'myfont' ;
 }

 .diyfont{
    font-family: 'myfont';
}

当然啦，这里几款字体的下载链接如下：

https://cdn.jsdelivr.net/gh/sky20681/sky20681/font/Aa望江南.ttf

https://cdn.jsdelivr.net/gh/sky20681/sky20681/font/告白体.ttf

https://cdn.jsdelivr.net/gh/sky20681/sky20681/font/Aa初吻.ttf

https://cdn.jsdelivr.net/gh/sky20681/sky20681/font/Aa奶糖初恋体.ttf

https://cdn.jsdelivr.net/gh/sky20681/sky20681/font/Aa甜茶.ttf

TIPs：我是不支持字体有版权的，原因很简单！

新建完CSS后，需要引用他

在/themes/matery/layout/_partial/head.ejs文件中插入引用，由于还没有远程部署，我们推导出它的http链接：

<link rel="stylesheet" type="text/css" href="https://cdn.jsdelivr.net/gh/sky20681/sky20681/css/diy.css">

也可以部署一下，然后打开链接看能否正常显示

引用完后，需要更改生成样式

在themes/matery/中，查找font-family参数

更改后面的值为diyfont，如图：

基本完成

最后，hexo cl
hexo g
hexo deploy
把需要部署的资源全部上传

打开链接测试链接的准确性

不出意外，推送后，打开网站并没有什么改变
这是因为CDN缓存的神奇

这时候，需要注销掉CDN缓存,在本地查看效果

# CDN访问加速 
# 第一次使用本功能，一定要先配置url，再`hexo cl && hexo g && hexo d`部署到GitHub的仓库，注意！必须是GitHub的仓库！
# 如果必须要使用国内的coding或者gitee，可以采用双部署，同时将网站部署到两个仓库（其中一个必须是GitHub的仓库）
# URL配置规则（例子如下）： https://cdn.jsdelivr.net/gh/你的GitHub用户名/你的仓库名
# 如果想关闭此功能，将 url地址 注释或删除即可！
#
# 注：配置了此项，就代表着本地调试的时候，网站依然会去GitHub请求资源（原来的资源），本地调试的时候记得将 此项配置 注释或者删除掉
# 更多关于访问速度优化，请查看：https://blog.sky03.cn/posts/42790.html
# 注：jsdelivr可以自动帮你生成.min版的js和css，所以你在上面设置js及css路径中可以直接写.min.xxx

hexo s启动本地端口查看

接着介绍更改字体大小

edge浏览器按F12，点击这个图标后，到网页上点击文字，看是哪个参数在调控它的大小，手动修改数值到满意大小，然后到本地文件上修改,找到font-face参数，把后面的值改为myfont

tips:
VS code,按F1搜索MArkdown插件可以实时查看，不用再下载专用的markdown编辑器了

为什么一套code，不同浏览器加载出不同的字体!!!???🤔️🤔️
